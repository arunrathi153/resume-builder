# 📄 Resume Builder Pro

A feature-rich, AI-powered resume builder with live preview, ATS scoring, multiple templates, and Firebase authentication.

🔗 **Live Demo:** [arunrathi153.github.io/resume-builder](https://arunrathi153.github.io/resume-builder)

---

## ✨ Features

- 🧠 **AI-Powered Summary** — Enhances your professional summary using Claude AI (Anthropic API)
- 📊 **ATS Score Meter** — Real-time animated score with tips to improve your resume
- 🎨 **3 Resume Templates** — Classic, Modern (sidebar), and Minimal
- 🎨 **Accent Color Picker** — 6 presets + custom color wheel
- 📸 **Profile Photo Upload** — Optional photo shown in resume preview
- 🔢 **8-Step Guided Wizard** — Personal → Template → Summary → Education → Skills → Projects → Achievements → Experience
- 👁️ **Live Preview** — See changes instantly with zoom in/out
- 🔗 **LinkedIn / GitHub / Portfolio** — Links shown in resume
- 🖨️ **Download as PDF** — Print-ready resume via browser print
- 🔐 **Firebase Authentication** — Login / Signup before download
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop

## 📋 How to Use

1. **Fill Personal Info** — Name, email, phone, location, social links
2. **Choose Template** — Pick Classic, Modern, or Minimal
3. **Write Summary** — Or click ✨ AI Enhance to auto-generate
4. **Add Education** — Degree, school, year, grade
5. **Add Skills** — Categories with quick-add buttons
6. **Add Projects** — Title, tech stack, bullet points, live link
7. **Add Achievements** — Certificates, awards, accomplishments
8. **Add Experience** — Internships, jobs with bullet points
9. **Download** — Login/Signup → Print as PDF

## 🔐 Authentication

This project uses **Firebase Authentication** for login/signup before resume download. Users are stored in Firebase — no resume data is saved to any server.
